# 🎓 Purdue-Colab-Hyobin-Park

Purdue 수업 실습(Lab)과 머신러닝 튜토리얼을 위한 Google Colab 노트북 모음입니다.

## 📁 Repository Structure

| 폴더 | 설명 |
|------|------|
| `lab/` | Raspberry Pi 기반 IoT 센서 실습 (Lab 0, Lab 1 등) |
| `ml_tutorial/` | 머신러닝 학습 튜토리얼 노트북 |

## 🛠️ Tech Stack

- Python
- Google Colab
- Raspberry Pi

## 📌 About

각 노트북은 SSH/VNC를 통한 원격 접속, IoT 센서 데이터 수집, 데이터 시각화 등의 실습 내용을 담고 있습니다.
